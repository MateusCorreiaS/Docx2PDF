# Docx2PDF

## Sobre a ferramenta

<strong>Docx2PDF</strong> é uma solução que permite o usuário, selecionar múltiplos arquivos <code>.docx</code> e transforma-los em arquivos <code>.pdf</code> sem limitações de uso e tamanho do arquivo.

O coração do Docx2PDF é o uso do <strong>LibreOffice</strong> para a conversão dos arquivos de forma consistente e o <strong>C# Windows Forms</strong> para interface.

## Depêndencias

- SDK .NET 10.0 ([Download](https://dotnet.microsoft.com/en-us/download/dotnet/10.0))
- LibreOffice ([Download](https://www.libreoffice.org/download/))


>A não instalação das dependências elencadas acima, irá ocasionar o <strong>não funcionamento</strong> do programa.
